 Данный проект составлен в целях самообразования и изучения Java Spring
 Для успешной компиляции необходимо заполнить поля в application.properties в папке resources
 
Для примера:
 
    spring.datasource.url=jdbc:postgresql://localhost/postgres
    spring.datasource.username=postgres
    spring.datasource.password=password

Также неопходимо исполнить файл build.gradle для загрузки всех внешних зависисостей
